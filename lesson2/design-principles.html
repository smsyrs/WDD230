White Space and Clean Design

Tenable

Tenable.com

This website shows a great use of the White Space and Clean Design Principles allowing their subpages to really work for them. The home page only offers the most pertinent information, as well as letting the visuals speak for words.

Repetition

GitHub

Github.com

GitHub's mobile viewport shows a great use of Repetition because of the nature of the website. This can be seen with a different color and font being used for the code than for the actions the developer is doing to navigate to that file. This allows the user to clearly idenitify the steps that are being taken.

Visual Heirarchy

Reddit

Reddit.com

Reddit's mobile viewport establishes a Visual Heirarchy by using a bigger font size for the information the user will care most about, in this case the title of the post, and continually uses smaller font sizes or a lighter color for the less desirable information.
